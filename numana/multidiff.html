Multivariate differentiation

Suppose that you have a function $f:{\mathbb R}^n\rightarrow{\mathbb R}.$ Then we say that it is differentiable at some $\v a\in{\mathbb R}^n$ if locally at $\v a$ the function may be "well approximated" by a linear function passing through $(\v a, f(\v a)).$ Such a function has the form $$ g(\v x) = f(\v a) + \v D' (\v x -\v a). $$ This linear function is what we call the tangent if $n=1$ and the tangent plane if $n=2.$ When this happens, $\v D$ is called the derivative of $f$ at $\v a.$

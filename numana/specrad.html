Spectral radius

Consider the function $f:{\mathbb R}\rightarrow{\mathbb R}$ given by $f(x) = ax,$ where $a\in{\mathbb R}.$ If we think of $x\in{\mathbb R}$ as an arrow (from 0 to $x$ along the numberline), then this function stretches (or shrinks and/or reflects) the arrow. What is the effect of this function on the length of the arrow? The answer is simple, it gets multiplied by $|a|$, i.e., $$ |ax| = |a|\cdot|x|. $$ We often need an analogous result for higher dimensions, i.e., when $f:{\mathbb R}^n\rightarrow{\mathbb R}^n$ is of the form $f(\v x) = A\v x.$ Here $A$ is an $n\times n$ matrix. Unfortunately, there is no direct analog if $n\geq 2,$ because even if $\v x$ and $\v y$ have the same length, i.e., $\|v x\| = \|v y\|,$ there is no guaranty that $\|A\v x\|$ will equal $\|A\v y\|.$

However, there is a partial analog: we can have an inequality. For any given $A$ there is a nonnegative number $\rho$ such that for all $\v x\in{\mathbb R}^n$ we have $$ \|A\v x\|\leq \rho \|\v x\|. $$ The smallest such $\rho$ is called the spectral radius of $A.$ It plays the role of the "absolute value" of a matrix.

It is quite easy to see why such a number exists. For this we shall employ a little fact from analysis:
Any continuous function defined over a closed and bounded set in ${\mathbb R}^n$ must attain its maximum somewhere.
Consider the set of all unit vectors in ${\mathbb R}^n.$ This is the unit circle in ${\mathbb R}^n.$ It is closed and bounded. Consider the function $F(\v x) = \|A\v x\|$ on this set. It is continuous (since $\v x\mapsto A\v x$ is continuous, and so is the $\|\cdots\|$ function). So it attains maximum. Take this maximum as $\rho.$ Do you see why this $\rho$ will serve our purpose?
